# Alexkk Blog

A Jekyll-based blog about braces care, oral hygiene, and smile confidence, with affiliate product recommendations.

🌐 **Live Site:** https://alexkk-ai.github.io/alexkk-blog

---

## About

**Alexkk Blog** (also branded as **Braces & Beyond by Alex**) publishes practical content for people with braces, including:

- Teeth whitening tips
- Gum health guidance
- Bad breath management
- Daily dental hygiene routines

The site is built as a static website using **Jekyll** and hosted in a GitHub Pages-compatible structure.

---

## Tech Stack

- **Jekyll**
- **Ruby + Bundler**
- **Minima theme**
- **Markdown + Liquid templates**

Plugins enabled:

- `jekyll-feed`
- `jekyll-seo-tag`
- `jekyll-sitemap`

---

## Run Locally

### Prerequisites

- Ruby
- Bundler (`gem install bundler`)

### Install

```bash
bundle install
```

### Start dev server

```bash
bundle exec jekyll serve --baseurl "/alexkk-blog"
```

Open: `http://127.0.0.1:4000/alexkk-blog/`
